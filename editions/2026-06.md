## Mid-June addendum

All 8 categories have been through initial landscape mapping. Key cross-category findings:

1. **MCP is the unifying protocol**: Every category has MCP integration emerging — vector DBs (MCP servers), code editors (MCP tool access), security (MCP gateway filtering), observability (MCP telemetry).
2. **OpenTelemetry is the unifying telemetry standard**: GenAI semantic conventions adopted across observability, model serving (GPU monitoring), and agent frameworks (span tracing).
3. **China's parallel ecosystem**: A fully independent domestic stack exists in every category — from Trae (code editors) to LMDeploy (model serving) to BISHENG (LLMOps) — with limited interoperability with Western protocols.
4. **Rust is infiltrating every layer**: Not just model serving (vLLM's C++ core, Rust inference engines) but agent frameworks (OpenFANG), vector DBs (Qdrant), and security (runtime firewalls).
5. **Consolidation is accelerating**: Every category has seen major M&A in 2025-2026 — from ClickHouse/Langfuse to Palo Alto/Protect AI to Cognition/Windsurf.
