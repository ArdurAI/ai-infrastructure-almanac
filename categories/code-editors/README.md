## Key Trends

- **Terminal-native agents winning**: OpenCode (150K+ stars), Gemini CLI, Claude Code, Kimi CLI — all terminal-first with generous free tiers.
- **Multi-agent parallel execution**: Google Antigravity Manager Surface, Claude Code Agent Teams, Kimi CLI Agent Swarm (300 sub-agents).
- **Pricing turbulence**: GitHub Copilot switched to usage-based credits June 2026. Cursor heavy users report $40–50/mo on $20 plan.
- **Security risks scaling**: 1 in 5 orgs had a serious security incident from AI-generated code. 45% of samples had OWASP Top 10 vulnerabilities.
- **China's domestic ecosystem**: Trae, 通义灵码, 文心快码, 豆包 MarsCode, 腾讯云AI代码助手 — fully independent, competitive on Chinese-specific tasks.
