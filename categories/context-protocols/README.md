## Key Trends

- **MCP security is critically immature**: 100% of ~2,000 scanned MCP servers lacked authentication; 30+ CVEs Jan-Feb 2026.
- **IETF standards urgency**: AIMS, WIMSE, SCIM for Agents drafted in Q1 2026 but 2-4 years from RFC status.
- **AI gateways becoming standard infrastructure**: 70% projected adoption by 2028.
- **Payment layer emerging**: UCP + x402/AP2/MPP building the commerce/payment layer for autonomous agents.

## Benchmarks

- **Standard**: MCP compliance test, A2A interoperability
- **Custom**: Protocol conformance, security posture, auth implementation correctness
- **Stress**: MCP server authentication bypass, tool poisoning, man-in-the-middle, revocation handling
