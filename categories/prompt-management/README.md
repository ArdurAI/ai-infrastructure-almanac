# Prompt Management

> **Cross-cutting concern.** Prompt management tools are currently tracked within the [LLMOps Platforms](../llmops-platforms/) category, as they are a core sub-function of the LLMOps lifecycle. This dedicated directory will be populated if the category grows large enough to warrant a split in a future edition.

## Key Tools (in LLMOps Platforms)

| Tool | Type | Focus |
|------|------|-------|
| PromptLayer | Prompt Management | Collaboration, testing, evaluation; no-code editor |
| Pezzo | LLMOps Platform | Prompt design, version management, troubleshooting |
| Vellum | Enterprise Orchestration | Experimentation, evaluation, deployment; A/B testing |
| Adaline | Prompt Management | End-to-end prompt lifecycle; versioning, testing, deployment |
| PromptHub | Prompt Management | Git-like workflow; versioning, branching, reviews |
| Humanloop | Prompt Management | Polished editing experience |
| Confident AI | Prompt Management | Git-style branching for parallel prompt experimentation |
| Promptfoo | Prompt Testing | Regression tests, prompt quality; CLI and CI/CD |
| PromptMage | Prompt Workflows | LLM workflows and prompts; self-hosted |
| PromptSite | Prompt Lifecycle | Version control, tracking, experimentation; lightweight Python |
| Prompteams | Prompt Management | Version, test, collaborate, retrieve; GitHub-style repos |
| Prompttools | Prompt Testing | Notebook-friendly; evaluate across different models |
| PromptDX | Prompt Development | Declarative, extensible, composable prompts; Markdown/JSX |
| PromptFoundry | Prompt Engineering | Simple prompt engineering and evaluation |
| Wordware | Agent IDE | Web-hosted IDE for task-specific AI agents |
| LangKit | Telemetry | LLM telemetry collection; features and profiles |
