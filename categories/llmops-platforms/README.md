## Key Trends

- **Evaluation-first replaced "ship and pray"**: Braintrust reports 30%+ accuracy improvements for customers within weeks.
- **No-code to code-first migration arc**: Teams prototype in Dify/Flowise/n8n, then migrate to code-first when hitting scale ceilings.
- **Chinese open-source platforms** (BISHENG, FastGPT, RAGFlow) gaining Fortune 500 traction.
- **Late-stage capital concentration**: 82.5% of disclosed funding goes to Series C+ rounds; median $230M.

## Benchmarks

- **Standard**: Custom workflow reliability, prompt regression
- **Custom**: Setup time, visual builder usability, prompt versioning correctness, export portability
- **Stress**: Workflow loop detection, concurrent workflow execution, node failure handling, cost-runaway prevention
